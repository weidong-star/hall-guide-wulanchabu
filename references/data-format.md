## 概览

每个政务服务大厅对应一个独立的JSON文件，文件命名为 `hall_{hallId}.json`（如 `hall_001.json`）。

JSON文件包含大厅的基本信息、窗口列表、办事事项、语音介绍文本等数据。

## 数据结构

```json
{
  "hallId": "001",
  "hallName": "市本级政务服务大厅",
  "address": "XX市XX区XX街道1号",
  "phone": "0471-1234567",
  "workTime": "周一至周五 9:00-17:00",
  "units": 25,
  "windows": [
    {
      "no": "1-10号",
      "floor": "1楼",
      "desc": "综合受理、咨询引导、帮办代办"
    },
    {
      "no": "11-25号",
      "floor": "2楼",
      "desc": "不动产登记、房产交易、抵押查询"
    }
  ],
  "items": [
    {
      "id": "i001",

}
```

## 字段说明

### 必填字段

| 字段 | 类型 | 说明 | 示例 |
|------|------|------|------|
| `hallId` | string | 大厅ID，3位数字 | "001" |
| `hallName` | string | 大厅名称 | "市本级政务服务大厅" |
| `voiceText` | string | 语音介绍文本，100-200字 | "欢迎来到..." |

### 可选字段

| 字段 | 类型 | 说明 | 示例 |
|------|------|------|------|
| `address` | string | 大厅地址 | "XX市XX区XX街道1号" |
| `phone` | string | 咨询电话 | "0471-1234567" |
| `workTime` | string | 办公时间 | "周一至周五 9:00-17:00" |
| `units` | number | 入驻单位数量 | 25 |
| `windows` | array | 窗口列表 | 见下方 |
| `items` | array | 事项列表 | 见下方 |
| `notice` | string | 温馨提示 | "请携带有效身份证件" |
| `image` | string | 大厅实景图片路径 | "images/hall_001.jpg" |

### windows 数组

每个窗口对象包含：

| 字段 | 类型 | 说明 | 示例 |
|------|------|------|------|
| `no` | string | 窗口编号 | "1-10号" |
| `floor` | string | 所在楼层 | "1楼"、"2楼" |
| `desc` | string | 窗口描述 | "综合受理、咨询引导" |

### items 数组

每个事项对象包含：

| 字段 | 类型 | 说明 | 示例 |
|------|------|------|------|
| `id` | string | 事项ID | "i001" |
| `name` | string | 事项名称 | "身份证办理" |
| `materials` | array | 所需材料 | ["户口本", "照片"] |
| `process` | string | 办理流程 | "取号→窗口办理→缴费→领取" |
| `time` | string | 办理时限 | "5个工作日" |
| `fee` | string | 办理费用 | "20元" |

## 验证规则

### 文件命名规则

- 格式：`hall_{hallId}.json`
- 示例：`hall_001.json`、`hall_012.json`
- `hallId` 必须为3位数字，不足补0

### 数据验证规则

1. **hallId**：必须是3位数字字符串（"001"-"999"）
2. **hallName**：不能为空
3. **voiceText**：不能为空
4. **units**：必须为正整数（如果存在）
5. **windows**、**items**：必须为数组（如果存在）

### 图片路径规则

- 格式：相对路径，如 `images/hall_001.jpg`
- 支持格式：jpg、jpeg、png、webp
- 推荐尺寸：800×400像素或以上

### 语音文本规则

- 长度：建议100-200字
- 内容：包含大厅基本信息、服务亮点
- 标点：使用中文标点符号

## 常见问题

### Q1: 如何新增一个大厅？

1. 创建新的JSON文件，命名为 `hall_{id}.json`（如 `hall_013.json`）
2. 填写大厅数据
3. 运行 `python scripts/generate-qr-urls.py --hall-count 13` 更新二维码列表

### Q2: 如何更新大厅信息？

直接编辑对应的JSON文件，保存后重新部署即可。

### Q3: 语音播报不工作怎么办？

1. 检查 `voiceText` 字段是否填写
2. 确保浏览器支持 Web Speech API（Chrome、Safari、Edge 均支持）
3. 检查设备是否开启了静音模式

### Q4: 图片不显示怎么办？

1. 检查图片路径是否正确
2. 确保图片文件已上传到 `images/` 目录
3. 检查图片文件名是否与 `image` 字段一致

### Q5: 如何添加多语言支持？

1. 为每个语言创建独立的JSON文件，如 `hall_001_zh.json`、`hall_001_mn.json`
2. 在页面添加语言切换功能
3. 根据选择的语言加载对应的JSON文件
